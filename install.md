# Installing MongoDB

* Install Homebrew: http://brew.sh/
* `brew install mongodb`
* `sudo mkdir -p /data/db` - then type in your password
* `sudo chown $USER /data/db`

## Start MongoDB Server

* `mongod`

## Install RoboMongo

* https://robomongo.org/download
